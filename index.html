💡 Reflexão e Aplicação Prática

🔧 Propriedades CSS Utilizadas

  • display: flex – transforma o container em um flex container.
  • justify-content – alinha os itens no eixo principal.
  • align-items – alinha os itens no eixo cruzado.
  • flex-direction – define a direção do eixo principal (linha ou coluna).
  • align-self – permite sobrescrever o alinhamento de um item individualmente.
  • flex-wrap – permite que os itens quebrem linha quando necessário.

🧠 Caso Real: Menu de Navegação

Em um menu de navegação horizontal, eu usaria Flexbox para alinhar os itens com `justify-content: space-between` para distribuir igualmente os links. Também usaria `align-items: center` para centralizar verticalmente os itens dentro do cabeçalho. Se fosse um menu responsivo, combinaria com `flex-wrap: wrap` para quebrar os itens quando a largura da tela for reduzida.
